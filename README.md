# listadetarefas.com.br
[Link da pagina na web.](https://listadetarefas.com.br/)

## Página web desenvolvida com a intenção de aprender e utilizar TypeScript.

## Observações:

### A pagina foi desenvolvida com fim acadêmico para aprendizado e treino com a linguagem TypeScript

### A pagina é totalmente Client-Side

## Objetivo da Página:

### 1- Inserir tarefas e criar uma lista

### 2- Iniciar a Tarefa desejada

### 3- É permitido pausar, excluir ou alterar os dados da tarefa a qualquer momento.

### 4- Finalizar a tarefa gerando uma lista com todas as tarefas concluidas e tempo dedicado as mesmas.
